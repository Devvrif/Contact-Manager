# Contact Manager

A simple command-line contact manager built in Python while learning fundamentals.

## Features

- ✅ Add new contacts (name + phone)
- ✅ View all contacts
- ✅ Search contacts by name
- ✅ Persistent storage (saves to file)

## What I Learned

- Python functions and parameters
- File I/O operations (read/write/append)
- Basic error handling with try/except
- Debugging type errors and syntax issues

## How to Run
```bash
python contact_manager.py
```

## Usage

1. Choose option 1 to add a contact
2. Choose option 2 to view all contacts
3. Choose option 3 to search for a contact
4. Choose option 4 to exit

## Technologies

- Python 3.x
- File I/O (no external libraries)

✅ Complete - Built in Week 1 of learning Python
